# 🚀 Antigravity Project Scaffolder

Bu araç, modern yazılım projelerini (Web, Mobile, AI/ML, Blockchain vb.) saniyeler içinde, profesyonel klasör yapılarıyla ve **Antigravity Ajan Seti** ile birlikte doğrudan masaüstünüze inşa eder.

## 🛠️ Kurulum

Arkadaşım, projeyi kullanmak için bilgisayarında **Node.js** kurulu olmalıdır.

1.  Projeyi indirin veya clone'layın.
2.  Terminali proje klasöründe açın ve bağımlılıkları yükleyin:
    ```bash
    npm install
    ```

## 🏃‍♂️ Çalıştırma

Bu uygulama iki parçadan oluşur: Arayüz ve Dosya Motoru. **İkisini de aynı anda** çalıştırmalısın:

1.  **Terminal 1 (Arayüz):**
    ```bash
    npm run dev
    ```
    *Bu komut arayüzü başlatır (genellikle http://localhost:3000).*

2.  **Terminal 2 (Dosya Motoru):**
    ```bash
    npm run backend
    ```
    *Bu komut, tarayıcıdan gelen komutları alıp Masaüstüne klasörleri yazan Node.js sunucusunu başlatır.*

## ✨ Özellikler

-   **11 Dev Şablon:** Web, Flutter, AI, Web3 ve daha fazlası.
-   **İnteraktif Editör:** Klasörleri aç/kapa, yeni dosya/klasör ekle, yanlışlıkla sildiğini geri al.
-   **Ajan Entegrasyonu:** Her proje `.agents/` klasörüyle (16 uzman rolü) birlikte doğar.
